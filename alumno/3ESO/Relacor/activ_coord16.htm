Ilusiones ópticas


Aquí tienes una serie de ejercicios para poner a prueba tu agudeza visual y, sobre todo, la capacidad de tu cerebro para hacerte ver cosas que no existen o para ocultarte las que existen.

1. ¿Qué animales hay?
2. El agua de una cascada, ¿siempre cae?
3. ¿Tumbado o levantado?
4. ¿Dos o tres puntas?
5. ¿Hay algo aquí?
