El control interno del organismo

1. Los sistemas encargados del control y regulación interno del organismo son: 

  1. El sistema circulatorio y el digestivo. 

  2. EL sistema endocrino y el sistema nervioso. 

  3. La sangre y la linfa.

  4. EL respiratorio y el circulatorio. 

  5. El sistema urinario y el respiratorio. 

2. En el sistema nervioso existen:

  1. Fibras musculares que recorren todo el cuerpo. 

  2. Fibras digestivas que recorren todo el cuerpo.

  3. Fibras nerviosas que recorren todo el cuerpo. 

  4. Fibras óseas que recorren el encéfalo.

  5. Fibras de colágeno que interconectan los órganos. 

3. En el sistema nervioso, la comunicación se establece mediante unos compuestos denominados: 

  1. Hormonas. 

  2. Neurotransmisores. 

  3. Enzimas. 

  4. Biocatalizadores. 

  5. Feromonas. 

4. El sistema endocrino está formado por: 

  1. Glándulas endocrinas conectadas entre sí.

  2. Glándulas exocrinas distribuidas por todo el cuerpo y que no están conectadas entre sí. 

  3. Glándulas endocrinas distribuidas por todo el cuerpo y conectadas entre ellas. 

  4. Glándulas exocrinas distribuidas por todo el cuerpo y conectadas entre ellas. 

  5. Glándulas endocrinas distribuidas por todo el cuerpo y que no están conectadas entre sí.

5. En el sistema endocrino, la comunicación se establece mediante unos compuestos denominados:  

  1. Hormonas. 

  2. Neurotransmisores. 

  3. Enzimas. 

  4. Biocatalizadores. 

  5. Feromonas. 

6. En el sistema endocrino, el transporte de los compuestos producidos por las glándulas:

  1. Lo realiza la linfa. 

  2. Lo realiza la orina. 

  3. Lo realiza la sangre. 

  4. Lo realiza el líquido sinovial. 

  5. Lo realiza el jugo pancreático.

7. A nivel celular, la acción del sistema endocrino se ejerce:

  1. Modificando la estructura celular. 

  2. Activando las mitocondrias 

  3. Activando o desactivando genes. 

  4. Actuando sobre los cloroplastos 

  5. Regulando la replicación de los cromosomas. 

8. El exceso de producción de las glándulas endocrinas se denomina:

  1. Hipofunción y puede producir determinadas enfermedades. 

  2. Regulación y no suele producir enfermedades. 

  3. Hiperfunción y no suele producir determinadas enfermedades. 

  4. Hiperfunción y puede producir determinadas enfermedades. 

  5. Hipofunción y no suele producir determinadas enfermedades. 

9. La acción del sistema endocrino se ejerce sobre órganos o tejidos denominados

  1. Diana 

  2. Receptores 

  3. Inhibidores 

  4. Reguladores 

  5. Coordinadores 

10. El sistema nervioso y el sistema endocrino

  1. Actúan de forma aislada. 

  2. No tienen ningún contacto entre ellos. 

  3. Actúan de forma independiente. 

  4. Actúan de forma antagónica. 

  5. Actúan de forma coordinada
